Como usar as Tags de Título <h1> a <h6>

(Voltar)

As tags de título vão de <h1> até <h6>. A <h1> é usada para o título principal da página, enquanto <h6> é para os subtítulos menos importantes. Essas tags ajudam a organizar o conteúdo, tornando a leitura mais fácil e estruturada. Pense no <h1> como o título do seu artigo e nos outros níveis como os subtítulos.

Essa tag, como a grande maioria das tags, requer "abertura" e "fechamento", como segue o exemplo:

<h1>Título 1</h1>
<h2>Título 2</h2>
<h3>Título 3</h3>
<h4>Título 4</h4>
<h5>Título 5</h5>
<h6>Título 6</h6>

Abaixo segue o exemplo prático dos tamanhos de títulos:

Como usar a Tag de Parágrafo <p>

(Voltar)

A tag <p> é usada para criar parágrafos. Qualquer texto que você quer separar do restante deve ser colocado dentro de <p> e </p>. Isso ajuda a formatar o texto com espaçamentos apropriados, deixando tudo mais legível.

Abaixo segue exemplo de utilização da tag <p>:

<p>Este é um paragráfo de exemplo</p>

Como usar a Tag de Marcação de Texto <mark>

(Voltar)

A tag <mark> é como usar um marca-texto para destacar partes importantes do texto. Ela adiciona um fundo amarelo ao texto selecionado, chamando a atenção do leitor para informações destacadas ou termos importantes.

Utilização da Tag:
<p>Esse é um exemplo da utilização da <mark>marcação de texto</mark> na prática.<p>

Resultado da aplicação da tag:
Esse é um exemplo da utilização da marcação de texto na prática.

Como usar a Tag <small>

(Voltar)

A tag <small> faz o texto ficar menor. É geralmente usada para notas de rodapé ou informações secundárias que não precisam chamar tanta atenção. Isso ajuda a manter o foco no conteúdo principal. No entanto, pode ser usado de outras maneiras, com aqui no site, que está sendo usado para apontar o link "voltar", que leva o leitor até o início da página.

<small>Usamos o small dessa maneira</small> e esse é o seu resultado: Usamos o small dessa maneira.

(Voltar)

A tag <strong> deixa o texto em negrito e indica que ele é importante. Diferente da tag <b>, que só altera a aparência, também sugere uma ênfase semântica, tornando o conteúdo mais acessível.

Como usar a Tag de lista ordenada <ol>

(Voltar)

A tag <ol> cria uma lista numerada, ideal para sequências que precisam seguir uma ordem específica, como instruções passo a passo. Cada item da lista é colocado dentro de uma tag <li>.

Utilização da Tag:
Lista de tarefas:
<ol>
<li>Ir ao banco</li>
<li>Sacar dinheiro</li>
</ol>

Resultado da aplicação da tag:
Lista de tarefas:
  1. Ir ao banco
  2. Sacar dinheiro

Como usar a Tag de lista sem ordem <ul>

(Voltar)

A tag <ul> cria uma lista com marcadores, usada para itens que não precisam seguir uma ordem específica, como uma lista de compras. Assim como a lista ordenada, cada item é colocado dentro de <li>.

Utilização da Tag:
Lista de tarefas:
<ul>
<li>Ir a academia</li>
<li>Ir ao supermercado</li>
</ul>

Resultado da aplicação da tag:
Lista de tarefas:
  • Ir a academia
  • Ir ao supermercado

Como usar a Tag de Item em lista <li>

(Voltar)

A tag <li> define cada item em uma lista, seja ela ordenada (<ol>) ou não ordenada (<ul>). Cada item da lista vai dentro de uma tag <li>, ajudando a organizar os itens de forma clara.

Exemplos da utilização de <li> encontram-se nos links Lista Ordenada e Lista Não Ordenada.

Como usar a Tag de Link <a>

(Voltar)

A tag <a> cria links, permitindo que os usuários naveguem entre páginas ou seções de um documento. O atributo href dentro da tag <a> especifica o destino do link, seja uma nova página, uma seção específica ou um endereço de e-mail.Em nosso site, por exemplo, estamos utilizando para linkar seções e o retorno ao início do site (índice), como você pode ver no termo (voltar) abaixo do Sub-título.

Utilização da Tag:
Clique no termo "início" e retorne ao ínicio do site: <a href="#inicio">início</a>

Resultado da aplicação da tag:
Clique no termo "início" e retorne ao ínicio do site: início

Como Usar o Atributo <href>

(Voltar)

O atributo href é usado dentro da tag <a> para especificar a URL de destino do link. Sem ele, a tag <a> não funciona como um link. Pode apontar para URLs externas, arquivos, ou seções da mesma página.Em nosso site, por exemplo, estamos utilizando para apontar para a seção índice do site, bem como do índice para as tags e atributos abordados.

Utilização da Tag:
Clique no termo "início" e retorne ao ínicio do site: <a href="#inicio">início</a>

Resultado da aplicação da tag:
Clique no termo "início" e retorne ao ínicio do site: início

Como usar a Tag de Subscrito <sub>

(Voltar)

A tag <sub> faz o texto ficar subscrito, ou seja, aparece abaixo da linha normal do texto. É útil para fórmulas químicas ou anotações matemáticas, como no exemplo H<sub>2</sub>O.

Resultado da aplicação da Tag: H2O

Como Usar a Tag de Sobrescrito <sup>

(Voltar)

A tag <sup> faz o texto ficar sobrescrito, aparecendo acima da linha normal do texto. É ideal para expoentes matemáticos, como E = mc<sup>2</sup>.

Resultado da aplicação da Tag: E = mc2

Como Usar a Tag de Citação <blockquote>

(Voltar)

A tag <blockquote> é usada para citações longas de outros textos. Ela geralmente aplica um recuo ao texto, destacando a citação do restante do conteúdo. Ideal para citar parágrafos inteiros. Na criação dessa página do site, usei o "blockquote" somente com a intenção de destacar aplicações de exemplos.

Utilização da Tag:
<blockquote>

<p>Esssa é a estrutura de exemplo de uma citação simples </p>

</blockquote>

Resultado da aplicação da tag:

Esssa é a estrutura de exemplo de uma citação simples

Como Usar a Tag de Fonte <font>

(Voltar)

A tag <font> é utilizada para definir a aparência do texto, como cor, tamanho e tipo de fonte. Embora não seja mais recomendada em HTML5, ainda pode ser encontrada em códigos antigos. Hoje em dia, é melhor usar CSS para isso.

Alé disso, nesse mesmo tópico, incluirei a Tag Comentário <!-- texto aqui -->

que é utilizada para comentar o código e tornar mais fácil a compreensão dos passos executados. Os comentários não são impressos na tela do cliente final, sendo armazenados somente no código, para revisões e explicações ao desenvolvedor.
<!-- Cor do texto -->
<font color="blue">Este texto está em azul.</font>

Resultado da aplicação da tag:
Este texto está em azul.

<!-- Tamanho do texto -->
<font size="5">Este texto tem tamanho 5.</font>

Resultado da aplicação da tag:
Este texto tem tamanho 5.

<!-- Tipo da fonte -->
<font face="Arial">Este texto está usando a fonte Arial.</font>

Resultado da aplicação da tag:
Este texto está usando a fonte Arial.

<!-- Combinação de cor, tamanho e fonte -->
<font color="green" size="4" face="Verdana">Texto verde, tamanho 4, fonte Verdana.<font>

Resultado da aplicação da tag:
Texto verde, tamanho 4, fonte Verdana.

Como Usar a Tag de Texto Tachado <del>

(Voltar)

A tag <del> é usada para indicar texto que foi removido. O texto aparece tachado, mostrando que foi deletado. Útil em revisões de documentos ou para indicar alterações no texto.

A tag <del> também pode ter atributos como <datetime> para indicar quando a alteração foi feita, proporcionando mais contexto sobre a modificação.

Vejamos exemplos abaixo das aplicações:

Utilização da Tag Del:

<p>Este é um <del>texto deletado</del> em um parágrafo.</p>

Resultado da aplicação da tag:

- Este é um texto deletado em um parágrafo.
- O preço antigo era R$ 100,00, mas agora é R$ 80,00.

Utilização da Tag Del com Datetime:

<p>Este texto foi <del datetime="2024-06-10T14:00">deletado </del> em uma data específica.</p>
<p>O preço antigo era <del datetime="2024-05-15">R$ 100,00</del>, mas agora é R$ 80,00.</p>

Como Usar a Tag de Abreviação <abbr>

(Voltar)

A tag é usada para marcar abreviações ou acrônimos. Você pode adicionar um atributo title que fornece a explicação completa do termo, ao passa o mouse em cima do termo. Isso melhora a acessibilidade, dando mais contexto aos leitores.

Exemplo de como usar a Tag abbr:
<p> O <abbr title="HyperText Markup Language"> HTML </abbr> é a linguagem padrão para criação de páginas web.</p>

O HTML é a linguagem padrão para criação de páginas web.
